== Overview

This documentation is for the search operator for {product-title}. The search operator has four possible requests: create, query, delete and update. 


=== Version information
[%hardbreaks]
_Version_ : 2.9.0


=== URI scheme
[%hardbreaks]
_BasePath_ : /kubernetes/apis
_Schemes_ : HTTPS


=== Tags
